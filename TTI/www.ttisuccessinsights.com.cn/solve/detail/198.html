明星课程
QUALITY COURSE

目标对象：基层、中层、高层、储备管理者
课程人数：15-20人
课程形式：线下工作坊，1天
课程所需测评：Motivator激励因子测评


课程简介：

本课程的目的是关于如何由心而发地驱动个人的热情以至于他们愿意积极主动地往目标前进。“激励因子”在指导过程中扮演了关键角色。身为教练，教练需要探索，挖掘，分辨被指导者的内驱力，所以当被指导者遇到挫败，目标停滞不前时能够及时提醒开火的点。此外，我们亦会提供教练指导的基础技巧，予以练习，因而更好服务，发展被指导者。

 

课程收益：

l  识别授权和激励下属的机会，应用有效的授权与激励方法与手段，发挥整个团队的力量和能力取得更高绩效

l  学习如何将授权用作发展下属的重要手段之一

l  全方位思考激发下属和团队持续工作积极性的方法和手段，而非单一的物质奖励手段

l  辨识辅导的种类和学习高效的辅导方法与流程

l  灵活运用建设性反馈技巧给予与接收反馈，以及辅导下属的方法和技巧

 

课程特色：

   课前提供版权Motivator激励因子测评

   结合测评工具，更加具备针对性，根据下属内驱力制定激励策略


课纲内容：

模块一：开场

l  了解期望

l  流程与规则说明

模块二：了解简单的指导架构跟流程 - 指导阶梯

l  说明

l  承诺

l  能力

l  修正

l  精炼

模块三：指导阶梯-说明

指导关系跟被指导者在目标成功与否与“说明”过程有很大的关联。教练跟被指导者的关键点在于第一次接触时建立信任跟开放的空间。

l  “说明”的目的

l  说明的5大环节

l  每个环节的注意事项

模块四：指导阶梯-承诺

说明人类的6大激励因子。透过“激励因子”激发被指导者的承诺，热情地朝目标前进。

l  理论、实用 、唯美、社会、个人、传统

模块五：指导阶梯-修正

第一部分：GROW模型

l  目标跟行动计划达成共识，采取行动后，教练们会运用简单实际的GROW模版进行每一次的指导谈话，协助被指导者提高自我意识，看到自己需要调整的点。

第二部分：进度阶段

l  集中期与动力期。身为教练，若要让被指导者更有动力，提高自信克服困难跟挑战，需要有能力区分他们达标的阶段。“小目标小赢”的概念跟容易“太快设定太大的行动”的倾向。学会如何处理在两种状况中教练可能犯的错：

n  当被指导者“不”采取行动

n  当被指导者超越订定的小目标

第三部分：主动聆听

l  理论上，教练在指导谈话过程中应该少说多听（比例3:7）。聆听看似简单，却具有挑战性。为培养被指导者之独立思考能力，“主动聆听”是必备技巧。

n  建立关系

n  放下偏见

n  复述话语

n  换句话说

n  总结

n  认知感受

第四部分：反馈

l  提供简单的反馈模式降低对方抗拒心理，从最客观的角度听见，因而协助被指导者反思。当结合GROW模版时，它能协助指导过程不费力地达成目的。

模块六：总结
